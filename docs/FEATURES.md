# Features

## Dashboard Features

### 1. Revenue Analytics
- **Monthly Revenue Trend**: Line chart showing revenue patterns over time
- Real-time data visualization
- Smooth scrolling animations

### 2. Customer Management
- **Top Customers Table**: Searchable and sortable customer list
- Churn status tracking (Active/Churned)
- Region and spending analysis
- Advanced filtering capabilities

### 3. Category Performance
- **Bar Chart**: Category-wise revenue breakdown
- Performance metrics visualization
- Comparative analysis

### 4. Regional Analysis
- **Regional Cards**: 6-region performance grid
- Key metrics: customers, orders, revenue
- Average revenue per customer tracking

## UI/UX Features

### Design
- 🎨 Modern glassmorphism design
- 🌙 Dark theme with neon gradients
- 📱 Fully responsive layout
- ✨ Smooth animations with Framer Motion

### Interactions
- 🖱️ Hover effects on cards and charts
- 📜 Scroll animations for content
- 🔍 Real-time search functionality
- ↕️ Multi-column sorting

### Data
- 📊 1000+ customer records
- 📈 5000+ transaction records
- 🗂️ 10+ product categories
- 🌍 6 regional segments

## Technical Stack

### Frontend
- React 18 with TypeScript
- Tailwind CSS for styling
- Recharts for data visualization
- Framer Motion for animations
- Vite as build tool

### Backend
- Python Flask
- CSV-based data storage
- RESTful API architecture
- CORS support

## Performance

- ⚡ Sub-second data load times
- 🔄 Efficient data processing
- 💾 Optimized bundle size (~745KB)
- 📦 Code splitting ready
